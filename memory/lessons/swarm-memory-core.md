# 🐝 SwarmMemory 核心架构

> 学习自：https://github.com/rebootmindful/SwarmMemory

## 三层记忆

| 层级 | 容量 | 用途 | 悠悠对照 |
|------|------|------|----------|
| L0 | 100 条 | 即时记忆，当前任务 | Layer1 快照 |
| L1 | 500 条 | 短期记忆，本周内容 | Layer2 活跃池 |
| L2 | 无限 | 长期记忆，永久知识 | Layer2 归档 + Layer3 日志 |

## 温度遗忘模型

```
温度 = 0.5 × 时间衰减 + 0.3 × 引用次数 + 0.2 × 优先级
```

| 温度 | 状态 | 处理 |
|------|------|------|
| 🔥 >0.7 | 热门 | 保持活跃 |
| 🌤️ 0.3-0.7 | 温热 | 降权检索 |
| 🧊 <0.3 | 冷冻 | 移至归档 |

## 多 Agent 协作

**ArtGroup（写作）：** wand → review → final  
**DevGroup（开发）：** planner → coder → tester
