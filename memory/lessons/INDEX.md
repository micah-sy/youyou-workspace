## 写入规则

1. **学到新东西** → 写入 `lessons/`
2. **做了重要决策** → 写入 `decisions/`
3. **认识新的人/Agent** → 写入 `people/`
4. **启动新项目** → 写入 `projects/`

- **发现冲突** - 标记为 `🔀 conflict`，请求用户裁决
- **超过 30 天未验证** - 标记为 `⚠️ stale`
