## 🤖 24/7 主动式 Agent

**memU 的核心创新：** 不是等用户命令，而是**主动理解用户意图**。

### 工作流程

```
┌─────────────────────────────────────────────────────────────────┐
│                         USER QUERY                               │
└─────────────────────────────────────────────────────────────────┘
                 │                                                   │
                 ▼                                                   ▼
┌──────────────────────────────┐         ┌──────────────────────────────┐
│      🤖 MAIN AGENT           │         │         🧠 memU BOT          │
│                              │         │                              │
│  处理用户查询 & 执行任务      │  ◄───►  │  监控、记忆、主动智能        │
├──────────────────────────────┤         ├──────────────────────────────┤
│                              │         │                              │
│  1. 接收用户输入              │         │  1. 监控输入/输出            │
│  2. 规划 & 执行               │  ◄───  │  2. 记忆 & 提取               │
│  3. 响应用户                  │         │  3. 预测用户意图             │
│  4. 等待下一轮                │  ◄───  │  4. 运行主动任务              │
└──────────────────────────────┘         └──────────────────────────────┘
                 │                                                   │
                 └─────────────────────┬─────────────────────────────┘
                                       ▼
                            ┌──────────────────────┐
                            │   持续同步循环        │
                            │ Agent ◄► memU ◄► DB │
                            └──────────────────────┘
```

### 悠悠的启发

悠悠目前是**被动响应**模式：
- 用户问 → 悠悠答
- 用户命令 → 悠悠执行

memU 的**主动式**模式：
- 监控用户行为 → 预测意图 → 主动提供帮助
- 例：用户在研究 AI 论文 → 悠悠主动推送相关新论文

**实现路径：**
1. 监控聊天记录（已实现）
2. 提取兴趣标签（待实现）
3. 定期搜索相关内容（待实现）
4. 主动推送（待实现）

## 💰 成本优化

**memU 的成就：** 将 24/7 在线 Agent 的 LLM token 成本降低到 ~1/10。

**方法：**
1. **缓存洞察** - 避免重复 LLM 调用
2. **小上下文** - 只加载相关记忆，不是全部历史
3. **后台监控** - memU Bot 独立运行，主 Agent 按需唤醒

**悠悠的对照：**
- ✅ 已实现：Consolidation 自动整理（减少上下文）
- ✅ 已实现：Layer1 快照（~2000 tokens）
- ⏳ 待实现：意图预测（减少无效对话）
- ⏳ 待实现：后台监控（独立 memU Bot）
