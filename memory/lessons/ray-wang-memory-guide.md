# 📚 Ray Wang 记忆系统实战

> 学习自：https://x.com/wangray/status/2027034737311907870

## 核心理念

> **文件 = 事实来源。你不写进文件的东西 = 你从来不知道的东西。**

## 三层架构

| 层级 | 文件 | 写入方式 | 用途 |
|------|------|----------|------|
| 工作台 | NOW.md | 覆写 | 当前状态、优先级 |
| 流水账 | memory/YYYY-MM-DD.md | 追加 | 事件日志 |
| 知识库 | lessons/, decisions/ | CRUD | 结构化知识 |

## 知识库健康度

```markdown
| 文件 | 优先级 | 状态 | 最后验证 | 说明 |
|------|--------|------|----------|------|
| [[cron-discipline]] | 🔴 | ✅ active | 2026-02-26 | 最近验证，可信 |
| [[legacy-setup]] | ⚪ | ⚠️ stale | 2026-01-15 | 42 天，可能过时 |
```

**设计原则：** 宁可暴露冲突，不可静默覆盖。

## 4 阶段渐进式构建

| 阶段 | 时间 | 新增内容 |
|------|------|----------|
| 阶段 0 | 立刻 | NOW.md + AGENTS.md + 每日日志 |
| 阶段 1 | 第 1 周 | INDEX.md + lessons/ + memlog.sh |
| 阶段 2 | 第 2 周 | Frontmatter + HEARTBEAT.md + 夜间反思 |
| 阶段 3 | 第 3 周 | QMD 语义搜索 + memory-gc.sh + 归档 |
| 阶段 4 | 第 4 周+ | 多 Agent 协作 + 自动过时检测 |
