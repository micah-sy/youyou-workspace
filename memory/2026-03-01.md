### 2. Scrapling 框架 🕷️

**来源：** https://github.com/D4Vinci/Scrapling

**定位：** 自适应网页爬取框架

**核心特性：**
- 自适应解析（网站改版后自动重新定位元素）
- 多种 Fetcher（StealthyFetcher 绕过 Cloudflare）
- Spider 爬虫框架（Scrapy 风格）
- 断点续传
- AI 集成（MCP Server）

**悠悠评价：** ⭐⭐⭐⭐⭐（4.8/5）
**适用：** 长期维护的爬虫项目、经常改版的网站

### 3. jina-cli 工具 📖

**来源：** https://github.com/geekjourneyx/jina-cli

**定位：** 为 AI Agent 打造的网页阅读 CLI

**核心功能：**
- `jina read` - 读取 URL 内容，输出 Markdown/JSON
- `jina search` - 网络搜索
- 支持 X/Twitter 阅读（解决悠悠之前的痛点！）
- 三种安装方式（OpenClaw Skill / Claude Code / CLI 二进制）

**悠悠评价：** ⭐⭐⭐⭐⭐（4.7/5）
**下一步：** 考虑安装为 OpenClaw Skill，增强网页读取能力
