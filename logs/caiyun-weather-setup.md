## 🔑 第 1 步：申请 API Key

### 1. 访问彩云开放平台

打开：https://platform.caiyunapp.com/

### 2. 注册/登录

- 可以用手机号注册
- 或者用微信/QQ 快捷登录

### 3. 创建应用

1. 登录后进入"控制台"
2. 点击"创建应用"
3. 填写应用信息：
   - **应用名称：** 悠悠天气（随意填）
   - **应用类型：** 个人项目
   - **描述：** 个人助手天气查询
4. 提交后获得 **API Key（Token）**

### 4. 复制 Token

格式类似：`d7c6f5e4a3b2c1d0e9f8a7b6c5d4e3f2`

## 🔧 第 2 步：配置到 OpenClaw

### 方式 1：告诉我 Token（推荐）

**直接告诉我：**
```
我的彩云天气 API Token 是：xxxxxxxxxxxxxxxx
```

我会自动帮你配置好！

### 方式 2：手动配置

**创建配置文件：**
```bash
cat > /home/admin/.openclaw/workspace/.caiyun-token << 'EOF'
你的 Token 在这里
EOF

chmod 600 /home/admin/.openclaw/workspace/.caiyun-token
```

**或者添加到环境变量：**
```bash
echo 'export CAIYUN_TOKEN="你的 Token"' >> ~/.bashrc
source ~/.bashrc
```

## 📊 API 使用示例

### 实时天气

```bash
TOKEN="你的 Token"
LOCATION="116.4,39.9"  # 经度，纬度

curl -X GET "https://api.caiyunapp.com/v2.6/$TOKEN/116.4,39.9/weather"
```

### 天气预报（3 天）

```bash
curl -X GET "https://api.caiyunapp.com/v2.6/$TOKEN/116.4,39.9/weather?dailysteps=3"
```

## ⚠️ 注意事项

### API 额度限制

| 类型 | 限制 |
|------|------|
| 每天调用次数 | 1000 次 |
| 每分钟请求数 | 10 次 |
| 每月免费额度 | 30000 次 |

**查看用量：** 登录控制台 → 用量统计

### Token 安全

- ✅ 保存在 `.caiyun-token`（权限 600）
- ✅ 不要提交到 Git（已在.gitignore 中）
- ✅ 不要公开分享
- ❌ 不要硬编码在脚本里

### 错误处理

| 错误码 | 含义 | 解决 |
|--------|------|------|
| 400 | 参数错误 | 检查坐标格式 |
| 401 | Token 无效 | 检查 Token 是否正确 |
| 403 | 额度用完 | 等待重置或升级 |
| 404 | 城市不存在 | 检查坐标 |
| 429 | 请求过多 | 降低频率 |
